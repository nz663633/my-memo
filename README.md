## My-Memo

A simple one-line memo project

---------

### Tech Stack

* **Frontend:** React.js
* **Backend:** Node.js, Express
* **Database:** MySQL

---------

### Key Features

* **User Authentication:** Sign-up and Login using Passport.js and bcrypt.
* **Personalized Data:"** Users can only manage their own memos through relational database mapping.
* **One-Line Memo:** Create, Read, and Delete short memos.
* **Responsive Design:** Clean and simple UI for all devices.
